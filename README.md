# What Time?
## Project Description

What time is a tool to organize events among google account users.

## Technologies
**Frontend:** ReactJs, Bulma
**Backend:** ExpressJs/NodeJs, PassportJs, GoogleAPI, Moongoose
**Database:** MongoDB
